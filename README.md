# itogGenerator
# Генератор случайных цитат

Этот проект представляет собой простой сайт-генератор случайных цитат. Сайт создан с использованием HTML, CSS и JavaScript, и предназначен для отображения случайной цитаты при нажатии кнопки.

## Особенности

- Случайная генерация цитат из заранее определенного списка.
- Возможность обновления цитаты по нажатию кнопки или клавиши Enter.
- Кнопка "Сделать репост", которая перенаправляет пользователя на страницу VK для репоста цитаты.

## Использование

Просто откройте сайт и нажмите кнопку "Сгенерировать цитату", чтобы увидеть новую цитату и её автора. При желании, вы можете поделиться цитатой на VK, нажав кнопку "Сделать репост".
